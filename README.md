<h1>Guide to running this program.</h1>
<p>1. run </p>

```
git clone https://github.com/SpeedSlicer/Ultraviolet-Static.git
cd Ultraviolet-Static
```
<p>2. Then run</p>

```
npm install
cd ..
```
<p>After that, run</p>



```
npm install ./Ultraviolet-Static
npm start
```
