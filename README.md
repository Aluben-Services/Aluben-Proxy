<h1>Guide to running this program.</h1>
<p>1. run </p>

```
git clone https://github.com/SpeedSlicer/Aluben-Static.git
cd Aluben-Static
```
<p>2. Then run</p>

```
npm install
cd ..
```
<p>After that, run</p>



```
npm install ./Ultraviolet-Static
npm start
```
